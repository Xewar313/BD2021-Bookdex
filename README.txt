Bookdex is application meant to keep track of your book collection, including books that you found interesting or simply want to read.

To use this application, you need to:
1. Install postgressql server
2. Create database with name "bookdex" and initialize it with Initdb.sql file
3. Create user with username "username" and password "password"

Each value can be changed in hibernate.cfg.xml file.

To start using it, first create new user in tab create and then select new user in dropbox on the left

After that application should be working properly.
